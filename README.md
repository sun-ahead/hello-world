# 🧮 简单计算器

一个适合初学者的 Python 项目，实现命令行计算器功能。

## ✨ 功能特点

- ✅ 支持加法、减法、乘法、除法
- ✅ 友好的命令行交互界面
- ✅ 中文提示，易于理解
- ✅ 包含错误处理（如除零检查）
- ✅ 代码结构清晰，适合学习

## 📁 项目结构

```
python-calculator/
├── calculator.py      # 主程序
├── test_calculator.py # 测试文件
└── README.md          # 项目说明
```

## 🚀 快速开始

### 环境要求
- Python 3.6 或更高版本

### 运行程序

```bash
# 进入项目目录
cd python-calculator

# 运行计算器
python calculator.py
```

### 使用示例

```
========================================
       🧮 欢迎使用简单计算器
========================================

请选择操作：
1. 加法 (+)
2. 减法 (-)
3. 乘法 (*)
4. 除法 (/)
5. 退出

输入选项 (1-5): 1
输入第一个数字: 10
输入第二个数字: 20
✅ 10.0 + 20.0 = 30.0
```

## 🧪 运行测试
